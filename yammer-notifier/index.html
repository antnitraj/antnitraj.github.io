Yammer Notifier

C:\> O Projektu _

Primjetio sam kako mi ne dolaze notifikacije sa Microsoftove platforme "Yammer" tako da sam odlučio napraviti sam svoj program sa kojim provjeravam ima li novih notifikacija. Bio je jako zanimljiv projekt za mene i jako sam puno naučio novih stvari.

C:\> Verzije Programa _

Koristim Yammerom built in API. Prva verzija koristi token koji možemo dobiti pomoću "Cookie editora" na Yammerovoj stranici (OPASNOST! Nemojte nikome dati taj token jer tada ima FULL ACCESS na vač AAIEDU račun!), Yammer Group ID te sa tim podatcima mogu koristiti Yammerov API da pošaljem GET Request na "/messages/in_group/GROUPID.json" API Endpoint, te sa tim podatcima izvadimo zadnju poslanu poruku, ako je ista kao i zadnja spremljena nastavljamo dalje, ali ako nije python skripta pokreće "sound.mp3" zvuk i ako smo stavili opciju "Auto Like" moj bot automatski like-a post u toj grupi. Za "Auto Like" pošaljem POST Request na Yammerov "/messages/liked_by/current.json" API Endpoint. Postupak se ponavlja svakih "Timeout" sekundi. Source Code je na mom Githubu GITHUB SORUCE CODE

Druga verzija je ista kao i prijašnja samo što sam ubacio da mi pošalje poruku na Discord Webhook što je jako korisno jer tada mogu dobiti tu notifikaciju makar sam bilo gdje u svijetu. GITHUB SORUCE CODE
